create-hyper-mendix-widget

“A CLI tool for creating hyper mendix widgets”

🚀 Hyper Mendix Widget??

A mendix widget that makes use of 1. dojo for passing widgets' properties and mounting the widget to DOM, & 2. Hyperapp v1.0 for building widget structure & functionality.

What is Hyperapp? 🤔

“Hyperapp is a JavaScript micro-framework (1.4kB) for building web applications.”
Minimal 👌 — The concepts you need to understand to be productive while remaining on par with what other frameworks can do are minimized.
Pragmatic 👨‍💻 — Hyperapp holds firm on the functional programming front when managing your state, but takes a pragmatic approach to allow for side effects, asynchronous actions, and DOM manipulations.
Standalone ✨ — Do more with less. Hyperapp combines state management with a virtual DOM engine that supports keyed updates & lifecycle events — all with no dependencies.

Global State

Set data-state="something" on a slide and "something" will be added as a class to the document element when the slide is open. This lets you apply broader style changes, like switching the page background.

State Events

Additionally custom events can be triggered on a per slide basis by binding to the data-state name. 


Reveal.addEventListener( 'customevent', function() {
	console.log( '"customevent" has fired' );
} );
